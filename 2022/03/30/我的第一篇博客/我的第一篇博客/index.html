 标题名
=============

# 标题名

在标题名下面用等号划线或者在标题名前加 #，注意 #号后面一定要加空格哦～

# 列表

# 有序列表

直接输入连续数字

1.  这是第一行列表项
2.  这是第二行列表项

# 效果

  1. 这是第一行列表项
  2. 这是第二行列表项

数字后要添加空格

# 无序列表

在列表项前面添加 * 号，+ 号 或是 - 号都可以

* Red
* Green
* Blue

+ Red
+ Green
+ Blue

- Red
- Green
- Blue

显示效果都是一样的

  • Red
  • Green
  • Blue

# 超链接

# 语法：

[超链接名称](超链接地址 "超链接的title")

举例：
     [欢迎查看小刘的博客](https://liuliuxiaoliu.github.io/ "liuliuxiaoliu")

# 效果

欢迎查看小刘的博客

# 定义超链接

# 语法：

[定义超链接名称]
[定义超链接名称]: 超链接地址 "超链接的title"
举例：
    [小刘的Blog]
    [小刘的Blog]:<https://liuliuxiaoliu.github.io/> "liuliuxiaoliu"

# 效果

小刘的 Blog

# title 可以不用写，定义好的链接可以在文件任意处给链接，建议放在文件尾部，便于修改

# 图片

# 语法

![图片名称](图片链接 "图片的关键词")
举例：
    ![我的头像](cover.jpg "liuliuxiaoliu")

# 效果

# 链接可以是网页链接，也可以是本地的链接，用绝对路径相对路径都可以
